# 🍽️ Food Munch Restaurant Website

A modern and responsive restaurant website that allows users to explore delicious food categories, discover healthy meal options, and learn about delivery services. Built with **HTML5**, **CSS3**, and **Bootstrap 5**, the project delivers an attractive and user-friendly interface across all screen sizes.

---

# 📖 Project Overview

Food Munch is a front-end restaurant website created to provide customers with an engaging online dining experience. The website highlights restaurant services, menu categories, healthy food promotions, delivery information, and customer engagement through a visually appealing and responsive design.

---

# ✨ Features

* Responsive navigation bar
* Attractive landing banner with call-to-action buttons
* "Why Choose Us" section highlighting restaurant services
* Multiple food menu categories
* Healthy food promotional section
* Delivery and online payment information
* Customer appreciation section with gift voucher modal
* Social media follow section
* Responsive footer with restaurant details
* Mobile-friendly design using Bootstrap Grid

---

# 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* Font Awesome Icons
* Google Fonts (Roboto)
* Bootstrap Grid System
* Responsive Web Design

# ⚙️ Installation Instructions

### Clone the repository

```bash
git clone https://github.com/your-username/Food-Munch.git
```

### Navigate to the project directory

```bash
cd Food-Munch
```

### Run the project

Open the `index.html` file in your preferred web browser.

Or use the VS Code Live Server extension for a better development experience.

```bash
Right Click → Open with Live Server
```
